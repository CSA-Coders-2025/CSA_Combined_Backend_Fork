About the Titanic Dataset

The Titanic dataset contains information about the passengers aboard the RMS Titanic, which sank after colliding with an iceberg on April 15, 1912. It includes details such as passenger class, age, gender, fare paid, and most importantly, whether they survived the disaster.

This dashboard presents various analyses of the data, showing how factors like gender, age, fare, and traveling companions correlated with survival rates.
